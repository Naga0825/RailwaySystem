# Railway Reservation System 🚆

A simple console-based **Railway Reservation System** built in Java.  
This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts, **Collections Framework**, and **Queue/ArrayList** for managing ticket reservations.

---

## 📌 Features
- **Book Ticket**: Adds passengers to confirmed list if seats are available, otherwise to waiting list.
- **Cancel Ticket**: Cancels a confirmed ticket and promotes the next passenger from the waiting list.
- **Waiting List Management**: Maintains a queue for passengers when seats are full.
- **Show Status**: Displays confirmed tickets and waiting list passengers.
- **Interactive Menu**: Console-based menu for booking, cancellation, and status checking.

---

## 🛠️ Tech Stack
- **Language**: Java  
- **Concepts Used**:
  - Classes & Objects
  - Encapsulation
  - Collections (`ArrayList`, `Queue`, `LinkedList`)
  - Control Structures (loops, switch-case)
  - Scanner for user input
